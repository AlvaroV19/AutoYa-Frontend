# AutoYa — Frontend 

Este repositorio contiene el Frontend Web PWA de AutoYa, un marketplace P2P para alquiler de vehículos entre propietarios y clientes. Enfocado en velocidad, experiencia de usuario y flujos intuitivos para reservas.

---

## Tecnologías principales

- Vue 3.3.4
- Axios
- Vite como bundler
- Vue Router
- Axios / Fetch para llamadas a microservicios
- Composición y componentes reutilizables

---

## Funcionalidades del Frontend

- Registro / Login (Cliente · Propietario).
- Publicación de vehículos con fotos.
- Búsqueda por ciudad, fechas y filtros (precio/tipo).
- Flujo de reserva con HOLD + confirmación.
- Módulo de contratos digitales.
- Panel del propietario para gestionar vehículos y reservas.
- Panel del cliente para revisar historial y próximas reservas.
